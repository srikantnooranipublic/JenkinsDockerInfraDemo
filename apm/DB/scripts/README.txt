createdb-postgres.sh creates a database called "timestock" on a
Postgres server.

createtables-postgres.sh creates and initializes the tables of
the Postgres database created by createdb-postgres.sh.
It reads createtables-postgres.sql and initdb-postgres.sql.

createtables-postgres.sql is generated by running
org.hibernate.tool.hbm2ddl.SchemaExport on the *.hbm.xml files,
but only the ones that correspond to database tables.
